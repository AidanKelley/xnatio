Installation
============

From Python Package Index
-------------------------
this is the recommended way to install xnatio

.. code:: bash

   pip install xnatio

From Source
-----------
clone the repository

.. code:: bash

   git clone https://github.com/AidanKelley/xnatio.git && cd xnatio

run setup.py

.. code:: bash

   python setup.py sdist install
